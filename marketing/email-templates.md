**Getting started in 3 minutes:**

1. Download PrepAura from your Gumroad purchase page
2. Install and open the app
3. Click ⚙ Settings → paste your license key → click Activate
4. Get a free Groq API key at **console.groq.com** → API Keys (no credit card)
5. Paste the Groq key in Settings and click Save

**Keyboard shortcuts:**
- `⌘⇧Space` — show/hide the overlay
- `⌘⇧C` — capture screen and solve a coding problem
- `⌘⇧A` — compact mode

---

**Tips for using PrepAura in interviews:**

- Set your **Interview Context** in Settings (e.g. "iOS Engineer at a Series B startup — Swift, UIKit, SwiftUI, system design") for more targeted answers
- Use **Mic + System audio** mode to capture both your microphone and the interviewer's voice over Zoom/Meet
- For coding challenges: press `⌘⇧C` to screenshot the problem and get an instant solution
